# LoginApp-Java

## Overview
LoginApp-Java is a simple web-based login application built using **Java Servlets, JSP, HTML, CSS, and MySQL**.  
It demonstrates a real-time login system where users can authenticate using credentials stored in a MySQL database.

This project is ideal for beginners learning Java web development and database connectivity.

---

## **Features**

- User authentication (login with username and password)
- MySQL database integration for storing user credentials
- Servlet-based backend
- JSP welcome page after successful login
- Modern, responsive login page using HTML and CSS
- Redirect to login
